## · eBook Style ·

- 标题
```css
h1 {
    font-size: 1.5em;
    text-align: center;
    color: #304090;
    margin-top: 1em;
    margin-bottom: 1.5em;
}
```
- 方框
```css
.tips {
    border: solid 2px #727272;
    border-radius: 5px;
    padding-left: 0.5em;
    padding-right: 0.5em;
}
```
```css
.tips {
    border-style: double;
    padding: 0.5em;
}
```
- 划线
```css
.underline {
    text-decoration: underline double 1px;
}
```
